Centos8安装Oracle19c

发表于2022-09-28|更新于2022-10-05|数据库
|阅读量:

前言

编写缘由

写这篇文章记录一下我那一段使用Centos8安装Oracle19c的头痛经历，前前后后折腾了两三天，甚至怀疑是Centos8安装出现了问题，而且老师给的教程又是比较老的版本，经过尝试百般错误，硬生生逼我速成Linux基本命令（以前搞过WSL，现在也在用，所以对于Linux(Ubuntu)有些许了解，但你不多），最终终于配置成功，由此记录一下，也希望能够对同样饱受折磨的同伴一些启发！

本文不一定完全适用于你，不合适或者报错的地方请自我查询，祝你好运

安装环境

Centos8(VmPro)+Oracle19c(采用rpm包安装)

Yum换源

如果你是现在第一次使用centos8，那么大概率你的yum已经无法正常使用了，因为该操作系统已经结束了它的生命周期（EOL），所以我们要换源，不了解的可以上网自行查询

操作的主要思路

在不删除源文件的基础上，将源文件备份，并下载最新centos8国内的yum源文件从而更换地址
主要实现的代码如下：

  • 进入root，切换至yum.repos.d目录
    cd /etc/yum.repos.d/

  • 创建新文件夹并将源文件备份为repo.bak
    mkdir backup && mv *repo backup/

  • 下载国内yum源文件
    curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-8.repo

  • 更新下载yum源地址
    sed -i -e"s|mirrors.cloud.aliyuncs.com|mirrors.aliyun.com|g " /etc/yum.repos.d/CentOS-*
    sed -i -e "s|releasever|releasever-stream|g" /etc/yum.repos.d/CentOS-*

  • 生成缓存
    yum clean all && yum makecache

由此大概率换源成功，因为每个人情况不一样，所以还需要看情况而定！

Oracle19c安装

下面我将结合两个教程教程1 和 教程2 并加之于我自己遇上的问题进行相关的完善！

1. 登录为root

之后不说切换用户则全程在该root用户且为 ~ (根目录)下面

输入命令 su

2. 下载OraclePreinstall和oracle19c两个rpm包

  • OraclePreinstall 下载安装有两个命令可以选择(选择其中一个即可)

    命令一：wget http://yum.oracle.com/repo/OracleLinux/OL7/latest/x86_64/getPackage/oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

    命令二：curl -o oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm https://yum.oracle.com/repo/OracleLinux/OL7/latest/x86_64/getPackage/oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

下载好了可以通过ls来进行查看是否存在这个文件
  • Oracle19c下载直接去官网选择下载这个就好了，登录之后就能下载了，下载好了之后先不用，你会直接安装也会报错，我们还需要其他操作才行

3. 安装oraclePreinstall

  • 输入以下命令进行

    先下载一些必要包：dnf -y install libnsl

    安装oraclePreinstall:yum -y localinstall oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

    你会发现输入这个命令会报错，显示缺少两个软件包compat-libstdc++和compat-libcap1，

你尝试用yum下载，但是错误提示显示并没有这两个包,马上进行解决：

  • 我们发现centos8没有这两个包，如果是centos7就有，但是箭在弦上不得不发了，我们可以直接下载centos7的包，是完全一样效果，compat-libstdc++和 compat-libcap1,
    可以通过该网址下载，并直接安装，下载之后用centos8(虚拟机环境)自带的软件安装工具直接可以进行安装，记住一定要安装二进制包，而非源文件包！

  • 同样的，CentOS Repositories可以下载我们需要的包，而自带的安装工具又可以极其方便安装或者可以选择用rpm安装命令进行安装

  • 之后安装oraclePreinstall:yum -y localinstall oracle-database-preinstall-19c-1.0-1.el7.x86_64.rpm

4. 安装 Oracle

  • 可以通过:yum -y localinstall xxxx.rpm安装
  • 或者是直接右键选择用软件安装方式打开进行安装即可

5. 初始化配置文件并创建Oracle SID实例

我们相关的配置文件为 /etc/init.d/oracledb_ORCLCDB-19c ,这是oracle给我们提供的样例，我么可以根据这个文件进行修改成为我们自己的配置文件

  • 配置文件中有一个很重要的参数，ORACLE_SID,我们要设置我们自己的，比如export ORACLE_SID=Sakura
  • 具体命令：
    • 复制一份样例 cp /etc/init.d/oracledb_ORCLCDB-19c /etc/init.d/oracledb_Sakura-19c
    • 进入修改 vi /etc/init.d/oracledb_ORCLCDB-19c /etc/init.d/oracledb_Sakura-19c
    • 找到相关ORACLE_SID 配置行进行修改 export ORACLE_SID=Sakura
    • ESC，:wq保存退出
    • 使其生效：/etc/init.d/oracledb_Sakura-19c configure

由此安装结束，希望大家顺利
