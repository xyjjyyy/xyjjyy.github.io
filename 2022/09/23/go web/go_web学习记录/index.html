1. 第一行是URL

  • http://localhost:8080/test 是网址
  • user=admin&password=123 是RawQuery,也就是我们请求传递的信息之一，我们可以通过 r.URL.RawQuery 来获得该字符串（string)，通常情况下 GET 请求是通过这种方式进行传递信息
  • 简便方法获得上Key-Value对：通过 Request 的 Form 字段，即：先使用 r.ParseForm()进行解析,然后可以得到r.Form()–>map[password:[123456] user:[admin]]
