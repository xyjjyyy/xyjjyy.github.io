前言

欢迎大家访问我的 个人网站，我以后会坚持更新有用的知识 ！

该网站采用 Hexo 并配置主题 butterfly，然后再在原有主题上进行自定修改，就是如今你们看到的样子啦 ！

本篇文章也就是和大家分享我自定义的过程，帮助大家也能自定义出好看的网站，嘿嘿 ！

同时本篇文章默认小伙伴们有一点前端的知识并且掌握了一下必要的工具的使用方法(eg:git npm)，这样才会更舒服的理解并自定义配置 ！

Hexo 安装及基本命令

Install hexo-cli

Create a new blog

注意格式是 `hexo init <博客的名字> 自己酌情替换即可

Create a new post

创建一篇文章，这是一种方式，也可以 ./source/_posts 目录下直接新建 markdown 文件并且按照格式附带相应的 Front-matter即可，具体写法可以进入官网网址学习

More info: Writing

Run server

执行这个命令将会在localhost:4000 进行本地服务，你可以在这种情况下，调试，预览

简洁版 hexo s
More info: Server

Generate static files

执行这个命令将会在根目录下生成 public 文件夹，用于储存生成的静态资源，同时我们部署网页的时候要用到，本地服务不需要使用该命令

简洁版：hexo g

More info: Generating

Deploy to remote sites

执行这个命令会进行部署，不过在部署之前还有一些配置要进行配置，详细见下文

简洁版：hexo d

More info: Deployment
